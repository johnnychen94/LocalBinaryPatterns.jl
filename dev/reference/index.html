LocalBinaryPatterns.lbp_original — Method
lbp_original(X)
lbp_original!(out, X)

Compute the local binary pattern, the original version [1], of gray image using 3x3 neighborhood matrix.

3x3 block     center-thresholded     weights         multiplied by weights      sum
6  7  9         1  1  1              1  8  32          1   8  32
5  6  3  ==>    0  x  0     ==>      2  x  64  ==>     0   x  0            ==>  169
2  1  7         1  0  1              4  16 128         0   0  128
julia> X = [6 7 9; 5 6 3; 2 1 7]
3×3 Matrix{Int64}:
 6  7  9
 5  6  3
 2  1  7

julia> lbp_original(X)
3×3 Matrix{UInt8}:
 0xc0  0x40  0x00
 0x68  0xa9  0x1b
 0x28  0x6b  0x00
